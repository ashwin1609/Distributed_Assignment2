# Distributed_Assignment2

## Run App

Installation 
Compile all the java files in both Client and Server folder

```
javac *.java
```
starting the rmiregistry in windows
```
start rmiregistry
```

starting the rmiregistry in Ubuntu/Solaris

```
rmiregistry&
```

Run the server 

```
java Server
```
Run the client 

```
java Client localhost
```
